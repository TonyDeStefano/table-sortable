# Sortable Table
Sort table columns with javascript

# Required Includes
- jquery
- font awesome

# Usage
Add class 'table-sortable' to any table. Requires correct usage of `<thead>`, `<tbody>` and `<th>` tags.

```html
<script src="table-sortable.js"></script>

<table class="table-sortable">
    <thead>
        <tr>
            <th>Column 1</th>
            <th>Column 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
        </tr>
        <tr>
            <td>Data 3</td>
            <td>Data 4</td>
        </tr>
    </tbody>
</table>
```
